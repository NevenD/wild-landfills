# Vue Web Map

### `Description`
Demo WEB GIS App for registering Wild LandFills. 

* Used tools:
    * https://openlayers.org/
    * https://cli.vuejs.org//
    * https://openlayers.org/
    * https://vuetifyjs.com/en/
 


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
